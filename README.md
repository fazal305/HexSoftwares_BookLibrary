# 📚 HexSoftwares Book Library Web App

A clean, responsive Book Library Web App built for the HexSoftwares Internship using **HTML5, CSS3, and Vanilla JavaScript**.

---

## 🚀 Features

### 🔍 Book Search
- Real-time search by title, author, or category
- Case-insensitive filtering
- Instant results update

### 🏷️ Categorization System
- Books grouped into categories:
  Fiction, Non-Fiction, Science, History, Technology, Biography, Self-Help, Other
- Category filter pills
- Combined search + category filtering

### 📚 Borrowing System
- Borrow books with borrower name
- Return books with history tracking
- Stores borrow & return timestamps

### 📜 Borrowing History
- Tracks all past borrow activities
- Shows borrower, dates, and duration
- Persistent history using localStorage

### 📊 Statistics Dashboard
- Total books count
- Borrowed vs available books
- Finished reading stats
- Live updates

### 💾 Data Persistence
- Uses browser localStorage
- No backend required

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Dark Cyberpunk UI)
- Vanilla JavaScript (ES6)
- localStorage API
